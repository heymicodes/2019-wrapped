Ce projet est basé sur le Year Wrapped de Spotify. Le but est de rassembler des données sur la consommation de musicale de l'utilisateur et le les retransmettre de manière digeste et colorée.

J'adore ce concept et j'ai voulu faire la même chose pour l'année 2019. Version développement web bien évidemment !

En raison du temps, le projet est statique mais j'ai dans l'idée de réitérer l'expérience chaque année.

Pour 2020, j'aimerais récupérer les données via l'API de gitHub et m'amuser plus avec javascript !

Je m'appelle Amy (ou Heymi pour certains).
